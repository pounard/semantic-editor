Todo-list

  • provide transformation tool: from p to ul, from ul to ol, from ul to set of p, etc...
  • when hitting delete key on last offset, parent item content should be moved to this item content
  • when hitting backspace key on first offset, current text should be moved to parent element
  • when hitting enter key in the middle, current text after should be split and moved to a new next element
  • enter key in h* should switch to a p under,
  • enter key in p should create a new p under,
  • enter key in li should create a new li under,
  • enter key in dd should create a dt under,
  • enter key in dt should create a dd under,
  • inline element should be underlined only, border have weird side effects,
  • blockquote should behave like an editor by itself,
  • on hover and left click, popup with allowed nested element should be created,
  • when typing or clicking on a non editable element, popup should go away,
  • set a column, on side, which displays pictograms of what's aside, fixed by theming
  • prevent focusing out the editor (optional, accessibility),
  • title text (accessiblity) should tell what enters does in element,
  • what is possible to do should be a in ruleset, somehow,
  • make it pluggable (medias, embed code, iframes), already partially is, more to come
  • when selection, provide a bubble/popup that gives possible insertions/alterations,
  • whichever happens, editor should always enforce typographic rules (no manual line feed, no empty paragraph).
